本プログラムは顔認証の登録、認証、登録削除をそれぞれ行うプログラムです。
※必ず内蔵カメラ搭載のPCもしくはUSB接続のWebカメラを接続した状態で実行してください。

インストールしておくべきライブラリ=====================================

・Tensorflow
・opencv-python
・numpy
・PySimpleGUI

インストールは
pip install tensorflow
pip install opencv-python
pip install numpy
pip install pysimplegui
をそれぞれ実行してください。

それでも起動しない場合はpipのバージョンを確認し、古い場合は
pip install --upgrade pip
でpipを更新し、再度ライブラリをインストールしてください。

ディレクトリ内の説明=============================================

・main.py
>>カメラに検出された候補をDB内の顔情報と合致するかを判定します。


・db_add.py
>>撮影した写真からAIが特徴を抽出し、DBに登録します。


・db_delete.py
>>DBに登録しているデータを削除します。（※CSVファイルは必ずここから削除してください）


・config.ini
>>必要な定数、登録情報をハッシュ化したデータを保管しています。


・data(ディレクトリ)
>>特徴データをCSV形式で保存しています。


・model(ディレクトリ)
>>AIのモデルと重みが入っています。


・haarcascades(ディレクトリ)
>>顔を検出するためのカスケードファイルが入っています。


基本操作===================================================

・main.py
顔の検出、認証： カメラに顔を合わせると自動で行われます。

結果の出力: コンソールに出力されます。

終了： Escキーもしくは×ボタン。


・db_add.py
撮影： Enterキー（※枠線には顎からおでこあたりまでを入れると理想です。）

終了： Escキーもしくは×ボタン。


・db_delete.py
削除するファイルの選択： プルダウンからお選びください

終了： キャンセルボタンもしくは×ボタン。


Q&A====================================================

Q1，カメラが起動しません
A１,カメラには管理者権限でしかプログラムがアクセスできない場合や、そもそもカメラの使用をOSの設定が許可していない可能性があります。まずはプログラムがデバイスにアクセスできる設定になっているかご確認ください。

Q2，起動が遅いです
A2，パソコンのスペックによってはAIの読み込みに時間がかかり起動が遅くなることがあります。
